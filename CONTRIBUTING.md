# Mental Health Prediction

A comprehensive, production-ready machine learning project for mental health prediction with MLOps best practices, Flask API, and Docker deployment.

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/Muhammad-Farooq-13/mental-health-prediction.git
cd mental-health-prediction

# Install dependencies
pip install -r requirements.txt

# Run the MLOps pipeline
python mlops_pipeline.py

# Start the Flask API
python flask_app.py
```

## ✨ Features

- 🤖 Multiple ML Models (Random Forest, Gradient Boosting, SVM, etc.)
- 🔄 Automated MLOps Pipeline
- 🌐 RESTful API with Flask
- 🐳 Docker & Docker Compose Support
- ✅ Comprehensive Testing with Pytest
- 📊 Automated Visualizations
- 📝 Complete Documentation
