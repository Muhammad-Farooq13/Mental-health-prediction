# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-29

### Added
- Initial release of Mental Health Prediction project
- Complete MLOps pipeline for automated training and deployment
- Flask REST API for model serving
- Docker and Docker Compose configuration
- Comprehensive data preprocessing module
- Feature engineering capabilities
- Multiple ML models (Logistic Regression, Random Forest, Gradient Boosting, SVM, Naive Bayes, Decision Tree)
- Model evaluation and comparison framework
- Automated visualization generation
- Unit tests with pytest
- Complete documentation (README, API docs)
- Exploratory data analysis notebook
- Configuration management with YAML
- Logging and monitoring capabilities

### Features
- Data loading and preprocessing
- Missing value handling
- Categorical encoding
- Numerical feature scaling
- Feature selection and engineering
- Model training with cross-validation
- Hyperparameter tuning support
- Model evaluation metrics (accuracy, precision, recall, F1, ROC-AUC)
- REST API endpoints for predictions
- Batch prediction support
- Docker containerization
- Health check endpoints
- Model metadata tracking

### Documentation
- Comprehensive README with setup instructions
- API documentation
- Code documentation with docstrings
- Testing guidelines
- Deployment instructions
- Contributing guidelines
